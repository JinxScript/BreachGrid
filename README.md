# BreachGrid: Rainbow Six Siege Strategy Planner

A planning tool for Rainbow Six Siege players. It helps teams create and save game strategies using a simple map drawing system. This project is currently in early development.

## Design

![App Design](design/landingpage.png)

## Features (Coming Soon)
- Draw and mark up maps
- Save and organize strategies
- Plan attacks and defenses
- Coordinate operator placements

## Technologies
- Flutter (Dart)

## Getting Started

### Requirements
- Flutter SDK
- Git

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/breachgrid.git
   cd breachgrid

### Install dependencies
```bash
flutter pub get
````

### Check Flutter setup

```bash
flutter doctor
```

### Run the app

```bash
flutter run
```
